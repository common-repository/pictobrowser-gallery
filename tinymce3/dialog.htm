Follow these steps to have a Flickr Slideshow displayed within the blog:

If you wish to display a slideshow from a users pool of photos, you must obtain the Flickr user ID from www.idgettr.com. If you are displaying a users set of photos, then you can skip this step.

Enter the album ID of the flickr set that you'd like to have in your slideshow, for example:
http://www.flickr.com/photos/library_of_congress/sets/72157603624867509/

Follow these steps to have a Picasa Slideshow displayed within the blog:

Enter the user ID from the creator of the album you want to display.

Enter the album ID of the Picasa album set that you'd like to have in your slideshow, cou can obtain it via the rss-feed of the album, for example:
http://picasaweb.google.com/data/feed/base/user/math.krueger/albumid/5366212228902710833
